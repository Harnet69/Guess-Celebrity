# Know your staff
## For whom?
* Human Resources staff : easy onboarding of newbies
* newbies : unstressful remembering colleagues faces
* employee : reminding “what’s the name of that colleague?”

## Features
* cool UI/UX
* step by step increasing difficulty
* training mode
